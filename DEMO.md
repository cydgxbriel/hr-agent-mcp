# 🎬 Demo script — 3 minutes

**Persona:** Ana Souza, HR manager. **Scenario:** Monday morning, team review.

## 0:00 — Opening (side panel visible)
Show the chat and the empty "MCP Calls" panel. One line: "every response
shows the MCP tools called — integration transparency".

## 0:20 — Operational query
> "How were [employee]'s punches over the last two weeks?"

Point at the panel: `consultar_batidas` + arguments. Highlight late arrivals and
the incomplete punch in the response.

## 1:00 — Policy grounding (RAG)
> "And what's the lateness tolerance under our policy?"

Point at: `consultar_politica`, response cites the source (politica-de-ponto.md).

## 1:30 — Action with governance (the climax)
> "What's pending approval?" → "Approve adjustment 1, the system was down."

Show the **confirmation card** (human-in-the-loop): the agent does NOT write
without confirmation. Confirm → approved + audit trail.
Ask to approve again → "already processed" (audit trail working).

## 2:20 — Analytics (BigQuery)
> "Which team accumulated the most overtime per month?"

Point at: `analytics_rh` with the SQL generated by the LLM and validated by the
governance layer before touching BigQuery.

## 2:50 — Closing
Architecture in one sentence: "Streamlit → LangGraph → MCP → SQLite/FAISS/BigQuery;
ETL feeds the warehouse; every write has human confirmation and an audit trail."
